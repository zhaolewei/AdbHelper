参数说明
adb_path : ADB工具路径，如："E\:\\software\\android-platform-tools\\adb"。
ui_rate : UI界面缩放比例，控制界面大小,，方便全屏显示。
jump_rate : 跳跃系数，跳跃距离与手机屏幕的相对比值；跳跃距离偏大或者偏小是可以通过该值来校正。
	以下为常见机型跳跃系数的参考值：
	一般机型 480*850 系数值为 ：3.100
	一般机型 540*960 系数值为 ：2.732
	一般机型 720*1280 系数值为 ：2.099
	一般机型 720*1440 系数值为 ：2.099
	一般机型 1080*1920 系数值为 ：1.392
	一般机型 1080*2160 系数值为 ：1.372
	一般机型 1440*2560 系数值为 ：1.475
	华为荣耀 NOTE8 系数值为 ：1.04
	华为荣耀 V8 系数值为 ：1.07
	小米Max2 系数值为 ：1.5
	小米5 系数值为 ：1.475
	小米5s 系数值为 ：1.475
	小米5x 系数值为 ：1.45
	小米6 系数值为 ：1.44
	小米NOTE2 系数值为：1.47
	三星S7 系数值为 ：1.0
	三星S8 系数值为 ：1.365 （注意该机型需要在设置里【关闭曲面侧屏】）
	锤子pro2 系数值为 ：1.392
